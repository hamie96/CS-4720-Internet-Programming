Database in Python

Objectives

The first objective is to use Python database access methods to produce reports.

The second objective is to set up code that will support web services you will develop in later assignments. In light of this objective, please pay careful attention to the organization specified for the code.

Overview

This assignment will use the measurements example database as a subject. Several of the later assignments will be built on some components you create for this assignment.

The measurements example is described in the Measurements Example page

Use the file measures.sqlite for the database.

You will create two modules that provide data from the database and then create a script that uses those modules.

It will be easiest to put all your files in the root directory of the project. Make sure you reference the database file by a relative reference. If the path to your file mentions directories on your system (such as your user directory, for example) then it will not work on the testing system.

Detailed Requirements

Unit test

You can use the following unit test to test the two modules: test2.py. A similar test will be used as part of the grading process.

Evaluation for Grade

Your two module files and the report script will be copied into a project that is set up with the database file and a unit test. Your report script will be run from there and the output checked. Then the unit test will be run.

  • Columns should line up in the report
  • The average value column should show the correct number of decimal places
  • An area with no measurements should display the proper entry
  • There should be no extraneous characters

Your code will be examined.

  • One important point that will be checked is that the module db_utility does not access the database directly but, rather, use the functions in db_access.
  • Also, report script should not access the database directly, but, rather, use the functions in the two modules.
  • Formatting of the average values to two decimal places should use the string formatting features in the Python library. In particular, do not use mathematical rounding operations.
